# Idea Grid

##About

Idea Grid is a responsive grid system.

##Features

* 12 Columns

* Fully Responsive

* Mobile First

* Fluid


##Getting Started

**Link to your stylesheet**

```

<link rel="stylesheet" href="css/ideagrid.css">

```
**Structure layout as example below**

